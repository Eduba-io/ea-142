# GiveMove Landing Page

A bespoke single-page site built for Alejandro Bisi and the GiveMove team ahead of eMerge Americas 2026 (Booth SS25). The page uses GiveMove's own brand tokens (purple #6A2F8B, lime-yellow #ECFD62 accent, Rubik across the board, pill-shaped CTAs, soft decorative circles) so it reads like something Mutanto could have shipped. The pitch body frames the rehab-tracking app as the next build problem after the shipped hardware, splits it across the 60/30/10 orchestration layers, cites the Feeld-shaped scoped sprint, links the Interpretable Context Methodology paper, and routes to a 30-minute call with Matt Creamer (CRO). Eduba attribution lives only in the footer.
